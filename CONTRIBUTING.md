# Contributing

Thanks for helping improve this template collection!

## How to Contribute

### Submit a new template

1. Use the [template submission form](https://github.com/LensmorOfficial/trade-show-email-templates/issues/new?template=submit-template.yml) to propose a new template
2. Or open a pull request directly

### Fix or improve an existing template

1. Fork this repo
2. Edit the template file
3. Open a pull request with a description of what you changed and why

## Template Format

All templates should follow this structure:

```markdown
# Template Name

> **When to use:** Brief scenario description
> **Best timing:** When to send this email
> **Expected response rate:** X-X% (if known)

## Subject Line Options

1. `Subject line 1`
2. `Subject line 2`
3. `Subject line 3`

## Email Body

The email text with [Variable] placeholders.

## Variables

| Variable | Description | Example |
|:---|:---|:---|
| [First Name] | Recipient's first name | Sarah |

## Tips

- Advice for using this template effectively
```

## Guidelines

- Use `[Bracket Variables]` for any personalized fields
- Include at least 2-3 subject line options
- Keep email bodies under 150 words
- Include a Variables table explaining each placeholder
- Add practical tips at the bottom
- Write in a professional but conversational tone
